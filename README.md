# engworks-test-lumen

### Installing

```
git clone https://github.com/glisandro/engworks-test-lumen.git
```

```
cd engworks-test-lumen
```

```
cp .env-example .env
```

```
set full path to project in APP_DATABASE in .env file
```

```
composer update
```

```
set your virtual host
```

```
go to http://project.test
```
